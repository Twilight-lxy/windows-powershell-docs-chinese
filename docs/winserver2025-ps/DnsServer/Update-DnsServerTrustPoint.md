# Update-DnsServerTrustPoint

## 摘要
更新DNS信任锚区域中的所有信任点。

## 语法

```
Update-DnsServerTrustPoint [-Force] [-ComputerName <String>] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Update-DnsServerTrustPoint` cmdlet 用于更新域名系统（DNS）信任锚区域中列出的所有信任点。

## 示例

### 示例 1：刷新指定 DNS 服务器上的所有信任点
```
PS C:\> Update-DnsServerTrustPoint -ComputerName "ContosoServer01" -Verbose
```

此命令会更新名为 ContosoServer01 的 DNS 服务器上的所有信任点。
