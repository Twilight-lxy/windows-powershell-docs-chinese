# 硬件认证模块
## 描述
本参考资料提供了所有 Windows 硬件认证工具包（HCK）cmdlet 的描述和语法信息。这些 cmdlet 按照其名称开头所使用的动词字母顺序进行排列。

## HardwareCertificationCmdlets
### [将HwCertTestCollection导出为XML文件](./Export-HwCertTestCollectionToXml.md)
将测试集合导出到.xml文件中。

### [从XML文件导入硬件证书测试集合](./Import-HwCertTestCollectionFromXml.md)
从.xml文件中导入一个测试集合。

### [从包中合并硬件证书测试集合](./Merge-HwCertTestCollectionFromPackage.md)
将多个.hckx包文件中的测试结果信息合并到一个单独的测试集合中。

### [从XML文件合并硬件认证测试集合](./Merge-HwCertTestCollectionFromXml.md)
将多个测试集合文件中的测试结果信息合并到一个测试集合中。

### [New-HwCertProjectDefinitionFile](./New-HwCertProjectDefinitionFile.md)
创建一个HCK项目定义文件。

### [New-HwCertTestCollection](./New-HwCertTestCollection.md)
根据项目定义文件创建一个测试集合文件。

### [New-HwCertTestCollectionExcelReport](./New-HwCertTestCollectionExcelReport.md)
创建一份HCK测试总结报告。
