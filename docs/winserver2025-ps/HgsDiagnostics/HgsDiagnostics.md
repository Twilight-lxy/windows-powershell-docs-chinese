# HgsDiagnostics模块
## 描述
该参考资料提供了所有Host Guardian Service Diagnostics客户端cmdlet的描述和语法信息。下表按字母顺序列出了这些cmdlet。

## HgsDiagnosticsCmdlets
### [Get-HgsTrace](Get-HgsTrace.md)
收集并分析与受保护织物的运行相关的数据。

### [Get-HgsTraceFileData](Get-HgsTraceFileData.md)
获取与HGS诊断跟踪文件相关的元数据及其内容，这些文件记录在跟踪清单（trace manifest）中。

### [New-HgsTraceTarget](New-HgsTraceTarget.md)
创建一个 HGS Diagnostics 目标对象。

### [Test-HgsTraceTarget](Test-HgsTraceTarget.md)
验证是否可以打开到指定目标的远程 Windows PowerShell® 会话。
