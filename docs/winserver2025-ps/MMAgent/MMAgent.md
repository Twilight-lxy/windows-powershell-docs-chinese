# MMAgent 模块
## 描述
该参考资料提供了所有与内存管理代理（Memory Management Agent，简称MMAgent）相关的cmdlet的描述和语法。这些cmdlet按照其命令开头所使用的动词字母顺序进行排列。

## MMAgentCmdlets
### [Debug-MMAppPrelaunch](./Debug-MMAppPrelaunch.md)
在特定应用程序启动之前，通过触发其预启动过程来调试该应用程序，并使该应用程序退出调试模式。

### [ Disable-MMAgent](./Disable-MMAgent.md)
禁用应用程序启动时的预加载功能、操作记录器API功能、页面合并功能以及应用程序的预启动过程。

### [Enable-MMAgent](./Enable-MMAgent.md)
启用应用程序启动前的预加载功能、操作记录器API功能、页面合并功能以及应用程序的预先启动功能。

### [Get-MMAgent](./Get-MMAgent.md)
用于获取应用程序启动时的预加载状态、操作API的预加载功能、页面合并过程以及应用程序启动前的相关情况。

### [Set-MMAgent](./Set-MMAgent.md)
为操作记录器（Operation Recorder）API所记录的场景设置预取文件的最大数量。
