# 最佳实践模块
## 描述
本参考资料提供了所有与 `-` 相关的 cmdlet 的描述和语法信息。这些 cmdlet 按照其开头所使用的动词（即命令行为）的字母顺序进行排列。

## BestPracticesCmdlets
### [Get-BpaModel](./Get-BpaModel.md)
检索并显示系统中安装的所有BPA模型列表。

### [Get-BpaResult](./Get-BpaResult.md)
检索并显示针对特定模型最近进行的最佳实践分析器（Best Practices Analyzer，简称BPA）扫描的结果。

### [Invoke-BpaModel](./Invoke-BpaModel.md)
开始对计算机上安装的特定模型进行BPA（Business Process Analysis，业务流程分析）扫描。

### [Set-BpaResult](./Set-BpaResult.md)
排除或包含BPA扫描的现有结果，仅显示指定的扫描结果。
