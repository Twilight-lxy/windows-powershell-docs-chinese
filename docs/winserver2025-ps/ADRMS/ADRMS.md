# ADRMS模块
## 描述
这个主题简要介绍了用于在 Windows Server 2012 中进行部署的 Windows PowerShell cmdlet（命令脚本）。表格中的每个 cmdlet 都链接到了关于该 cmdlet 的更多详细信息。这些 cmdlet 可以帮助您使用 Windows PowerShell 安装、升级或卸载集群。

## ADRMS 命令行插件（Cmdlets）
### [安装 ADRMS](./Install-ADRMS.md)
配置新的 AD RMS 服务器部署。

### [卸载 ADRMS](./Uninstall-ADRMS.md)
删除现有 AD RMS 服务器部署的配置信息。

### [更新-ADRMS](./Update-ADRMS.md)
更新现有的 AD RMS 服务器部署。
