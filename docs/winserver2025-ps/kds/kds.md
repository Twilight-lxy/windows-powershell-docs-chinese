# kds模块

## 描述

这个参考资料提供了所有与密钥分发服务（Key Distribution Service，简称KDS）相关的cmdlet的描述和语法说明，并将这些cmdlet按字母顺序进行了排列。

## kds Cmdlets

### [Add-KdsRootKey](./Add-KdsRootKey.md)

为 Microsoft Group KdsSvc 在 Active Directory 中生成一个新的根密钥。

### [Clear-KdsCache](./Clear-KdsCache.md)

清除本地计算机的组键缓存。

### [Get-KdsConfiguration](./Get-KdsConfiguration.md)

从 Active Directory 中检索 Microsoft Group KdsSvc 的当前配置信息。

### [Get-KdsRootKey](./Get-KdsRootKey.md)

检索由 Microsoft Group KdsSvc 存储的所有根键值的对列表。

### [Set-KdsConfiguration](./Set-KdsConfiguration.md)

设置 Microsoft Group KdsSvc 的配置信息。

### [Test-KdsRootKey](./Test-KdsRootKey.md)

测试根键配置。
