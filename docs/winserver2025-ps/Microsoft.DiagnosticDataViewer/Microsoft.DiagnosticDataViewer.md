# Microsoft.DiagnosticDataViewer 模块
## 描述
Microsoft.DiagnosticDataViewer模块提供了用于查看和分析Windows设备发送的诊断数据的命令。

## 最低要求
该模块需要 Windows 10 的 17134 版本（1803）或更高版本。

## Microsoft.DiagnosticDataViewerCmdlets
### [禁用诊断数据查看功能]( Disable-DiagnosticDataViewing.md)
禁用诊断数据查看功能。

### [Enable-DiagnosticDataViewing](Enable-DiagnosticDataViewing.md)
启用诊断数据查看功能。

### [Get-DiagnosticData](Get-DiagnosticData.md)
获取该机器上传的历史Windows诊断数据。

### [Get-DiagnosticDataTypes](Get-DiagnosticDataTypes.md)
获取诊断数据类型标识符与其相应描述的映射关系。

### [Get-DiagnosticDataViewingSetting](Get-DiagnosticDataViewingSetting.md)
用于获取当前是否启用了诊断数据查看功能。

### [Get-DiagnosticStoreCapacity](Get-DiagnosticStoreCapacity.md)
获取当前的诊断数据存储容量。参数 \[-Size\] 以兆字节（MB）为单位返回诊断数据存储的容量；参数 \[-Time\] 以天为单位返回诊断数据存储的容量。默认的诊断数据存储容量为 1024 MB，默认的时间期限为 30 天。

### [Set-DiagnosticStoreCapacity](Set-DiagnosticStoreCapacity.md)
设置诊断存储的时间和容量。
