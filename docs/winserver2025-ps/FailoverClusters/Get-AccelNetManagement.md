# 获取 AccelNet Management 数据

## 摘要
启用加速网络管理（Accelerated Networking Management）的相关功能/设置。

## 语法

```
Get-AccelNetManagement
```

## 描述

检索AccelNet管理所支持的意图（即用户想要执行的操作或请求）。

## 示例

### 示例 1

```powershell
Get-AccelNetManagement
```

这个示例用于获取已启用的意图名称。
