# Get-ClusterGroupSet

## 摘要
获取集群中的组集（group sets）。

## 语法

```
Get-ClusterGroupSet [[-Name] <String[]>] [-IsGlobal <Boolean[]>] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [<CommonParameters>]
```

## 描述

`Get-ClusterGroupSet` cmdlet 可以获取集群中的所有组集。

## 示例

### 示例 1：获取集群中的所有组集合

```powershell
Get-ClusterGroupSet
```

此命令用于列出集群中的组集。
