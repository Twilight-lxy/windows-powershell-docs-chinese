# 启用 NetAdapterRss 功能

## 摘要
启用网络适配器上的RSS功能。

## 语法

### 按名称查找（默认值）
```
Enable-NetAdapterRss [-Name] <String[]> [-IncludeHidden] [-NoRestart] [-PassThru] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInstanceID
```
Enable-NetAdapterRss -InterfaceDescription <String[]> [-IncludeHidden] [-NoRestart] [-PassThru]
 [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject (cdxml)
```
Enable-NetAdapterRss -InputObject <CimInstance[]> [-NoRestart] [-PassThru] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Enable-NetAdapterRss` cmdlet 可用于启用网络适配器的接收端扩展功能（即 RSS 功能）。RSS 是一种可扩展性技术，它通过對传入数据包的头部进行哈希处理，将网络接收流量分配到多个处理器上。在 Windows Server 2003 (FirstRef_longhorn) 及后续版本中，如果没有启用 RSS 功能，所有网络流量都会被发送到第一个处理器上，这可能导致该处理器迅速达到满负荷状态，从而限制了网络的接收吞吐量。

## 示例

### 示例 1：在指定的网络适配器上启用 RSS 功能
```
PS C:\> Enable-NetAdapterRss -Name "MyAdapter"
```

这个命令会为名为“MyAdapter”的网络适配器启用RSS功能，并重新启动该网络适配器。

### 示例 2：在所有支持 RSS 的网络适配器上启用 RSS 功能
```
PS C:\> Enable-NetAdapterRss -Name "*"
```

此命令可启用所有支持RSS功能的网络适配器上的RSS功能。
