# Set-NetAdapterRdma

## 摘要
设置网络适配器上的 RDMA 属性。

## 语法

### 按名称排序（默认值）
```
Set-NetAdapterRdma [[-Name] <String[]>] [-IncludeHidden] [-Enabled <Boolean>] [-NoRestart]
 [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-PassThru] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### 按实例ID（ByInstanceID）
```
Set-NetAdapterRdma -InterfaceDescription <String[]> [-IncludeHidden] [-Enabled <Boolean>] [-NoRestart]
 [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-PassThru] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### InputObject (cdxml)
```
Set-NetAdapterRdma -InputObject <CimInstance[]> [-Enabled <Boolean>] [-NoRestart] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Set-NetAdapterRdma` cmdlet 用于设置网络适配器上的远程直接内存访问（RDMA）相关属性。RDMA 是一种技术，它允许网络适配器之间直接传输数据，而无需计算机的主要处理器参与该传输过程。这样一来，可以降低延迟并减少处理器的使用率。`Enable-NetAdapterRdma` 和 `Disable-NetAdapterRdma` cmdlets 也可以用于管理 RDMA 功能。

## 示例

### 示例 1：在指定的网络适配器上启用 RDMA
```
PS C:\> Set-NetAdapterRdma -Name "MyAdapter" -Enabled $True
```

这个命令用于在名为“MyAdapter”的网络适配器上启用RDMA（远程直接内存访问）功能。**Enable-NetAdapterRdma** 是执行此操作的推荐使用cmdlet。

### 示例 2：禁用指定网络适配器上的 RDMA 功能
```
PS C:\> Set-NetAdapterRdma -Name "MyAdapter" -Enabled $False
```

此命令将名为“MyAdapter”的网络适配器上的RDMA功能禁用。**Disable-NetAdapterRdma** cmdlet是执行此操作的首选cmdlet。
