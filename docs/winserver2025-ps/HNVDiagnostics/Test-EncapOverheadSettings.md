# 测试封装开销设置

## 摘要
测试网卡（NIC）是否支持封装开销（encapsulation overhead）。

## 语法

```
Test-EncapOverheadSettings [<CommonParameters>]
```

## 描述
`Test-EncapOverheadSettings` cmdlet 用于检测主机中的网络接口卡（NIC）是否能够正确支持数据封装过程中的开销处理。
