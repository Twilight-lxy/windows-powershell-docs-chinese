# 测试逻辑网络连接

## 摘要
用于测试逻辑网络连接是否正常。

## 语法

```
Test-LogicalNetworkConnection [<CommonParameters>]
```

## 描述
`Test-LogicalNetworkConnection` 这个 cmdlet 用于测试是否可以连接到某个逻辑网络。
