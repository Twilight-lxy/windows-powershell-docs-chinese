# Get-DhcpServerDatabase

## 摘要
获取与DHCP服务器服务的数据库相关的配置参数。

## 语法

```
Get-DhcpServerDatabase [-ComputerName <String>] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [<CommonParameters>]
```

## 描述
`Get-DhcpServerDatabase` cmdlet 可以获取与动态主机配置协议（DHCP）服务器服务的数据库相关的配置参数。

## 示例

### 示例 1：获取 DHCP 服务器服务数据库设置
```
PS C:\> Get-DhcpServerDatabase -ComputerName "dhcpserver.contoso.com"
```

这个示例用于获取名为dhcpserver.contoso.com的计算机上运行的DHCP服务器服务的数据库设置信息。
