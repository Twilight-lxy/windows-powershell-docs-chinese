# Get-DhcpServerv4Binding

## 摘要
获取绑定DHCP服务器服务的计算机上的IPv4接口信息。

## 语法

```
Get-DhcpServerv4Binding [-ComputerName <String>] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [<CommonParameters>]
```

## 描述
`Get-DhcpServerv4Binding` cmdlet 可获取计算机上所有与动态主机配置协议（DHCP）服务器服务绑定的 IPv4 接口。

## 示例

### 示例 1：获取绑定到 DHCP 服务器服务的计算机上的接口
```
PS C:\> Get-DhcpServerv4Binding -ComputerName "dhcpserver.contoso.com"
```

这个示例会获取名为dhcpserver.contoso.com的计算机上所有绑定了DHCP服务器服务的IPv4接口。
