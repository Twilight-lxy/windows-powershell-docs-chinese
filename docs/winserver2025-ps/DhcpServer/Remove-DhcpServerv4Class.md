# 删除 DHCP Serverv4 类

## 摘要
从DHCP服务器服务中删除IPv4供应商类别或用户类别。

## 语法

```
Remove-DhcpServerv4Class [-ComputerName <String>] [-Name] <String[]> [-Type] <String> [-PassThru]
 [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Remove-DhcpServerv4Class` cmdlet 用于从动态主机配置协议（DHCP）服务器服务中删除 IPv4 的供应商类或用户类。

## 示例

### 示例 1：删除一个用户类
```
PS C:\> Remove-DhcpServerv4Class -Name "User Class for Lab Computers" -Type User
```

这个示例会从DHCP服务器服务中删除用于实验室计算机的DHCPv4用户类别。

### 示例 2：删除一个供应商类
```
PS C:\> Remove-DhcpServerv4Class -Name "MSUCClient" -Type Vendor
```

此示例会从 DHCP 服务器服务中删除适用于 Microsoft 统一通信客户端的所有 DHCPv4 供应商类别（vendor classes）。
