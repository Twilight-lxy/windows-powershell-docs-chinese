# Set-DhcpServerv4FilterList

## 摘要
设置DHCP服务器服务中允许（allow）和拒绝（deny）MAC地址过滤列表的启用状态。

## 语法

```
Set-DhcpServerv4FilterList [-ComputerName <String>] [[-Allow] <Boolean>] [[-Deny] <Boolean>] [-PassThru]
 [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Set-DhcpServerv4FilterList` cmdlet 用于设置动态主机配置协议（DHCP）服务器服务中允许和拒绝 MAC 地址过滤列表的启用状态。

## 示例

#### 示例 1：在 DHCP 服务器服务上启用过滤列表功能
```
PS C:\> Set-DhcpServerv4FilterList -ComputerName "dhcpserver.contoso.com" -Allow $True -Deny $True
```

这个示例为运行在名为dhcpserver.contoso.com的计算机上的DHCP服务器服务启用了“允许”（allow）和“拒绝”（deny）过滤规则列表。
