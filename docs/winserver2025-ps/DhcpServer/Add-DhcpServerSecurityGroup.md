# Add-DhcpServerSecurityGroup

## 摘要
将安全组添加到DHCP服务器中。

## 语法

```
Add-DhcpServerSecurityGroup [-ComputerName <String>] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>]
 [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Add-DhcpServerSecurityGroup` cmdlet 用于将安全组添加到动态主机配置协议（DHCP）服务器上。该 cmdlet 会添加 `DHCP Users` 和 `DHCP Administrators` 这两个安全组。

## 示例

### 示例 1：为 DHCP 服务器添加安全组
```
PS C:\> Add-DhcpServerSecurityGroup -ComputerName "DhcpServer03.Contoso.com"
```

此命令将“DHCP Users”和“DHCP Administrators”这两个安全组添加到名为DhcpServer03.Contoso.com的DHCP服务器上。
