# AppvSequencer 模块
## 描述
以下列表包含了指向 Microsoft 应用程序虚拟化（App-V）Sequencer cmdlet 帮助主题的链接。

## AppvSequencerCmdlets
### [Expand-AppvSequencerPackage](./Expand-AppvSequencerPackage.md)
用于扩展现有的App-V软件包。

### [New-AppvPackageAccelerator](./New-AppvPackageAccelerator.md)
生成一个包加速器（package accelerator）。

### [New-AppvSequencerPackage](./New-AppvSequencerPackage.md)
创建一个新的App-V软件包。

### [更新 AppvSequencerPackage](./Update-AppvSequencerPackage.md)
升级虚拟应用程序包。
