# BitLocker模块
## 说明
这个参考资料提供了所有 BitLocker 命令的功能描述和语法说明。这些命令是按照每个命令开头所使用的动词（即命令的操作类型）的字母顺序排列的。

## BitLocker cmdlets（BitLocker命令集）
### [Add-BitLockerKeyProtector](./Add-BitLockerKeyProtector.md)
为 BitLocker 卷添加一个密钥保护器。

### [BackupToAAD-BitLockerKeyProtector](./BackupToAAD-BitlockerKeyProtector.md)
在 Microsoft Entra ID 中为 BitLocker 卷保存一个密钥保护器。

### [Backup-BitLockerKeyProtector](./Backup-BitLockerKeyProtector.md)
在 Active Directory Domain Services (AD DS) 中为 BitLocker 卷保存一个密钥保护器。

### [Clear-BitLockerAutoUnlock](./Clear-BitLockerAutoUnlock.md)
删除BitLocker的自动解锁密钥。

### [禁用BitLocker](./ Disable-BitLocker.md)
禁用卷的BitLocker驱动器加密功能。

### [禁用 BitLocker 自动解锁功能](./ Disable-BitLockerAutoUnlock.md)
禁用BitLocker卷的自动解锁功能。

### [Enable-BitLocker](./Enable-BitLocker.md)
为某个卷启用 BitLocker 磁盘加密功能。

### [Enable-BitLockerAutoUnlock](./Enable-BitLockerAutoUnlock.md)
启用BitLocker卷的自动解锁功能。

### [Get-BitLockerVolume](./Get-BitLockerVolume.md)
获取有关 BitLocker 可以保护的卷的信息。

### [Lock-BitLocker](./Lock-BitLocker.md)
防止访问使用 BitLocker 加密技术保护的卷中的加密数据。

### [Remove-BitLockerKeyProtector](./Remove-BitLockerKeyProtector.md)
移除BitLocker卷的保护机制（即删除用于保护该卷的关键文件或设置）。

### [Resume-BitLocker](./Resume-BitLocker.md)
恢复指定卷的Bitlocker加密功能。

### [Suspend-BitLocker](./Suspend-Bitlocker.md)
暂停指定卷上的Bitlocker加密功能。

### [解锁 BitLocker](./Unlock-BitLocker.md)
恢复对使用 BitLocker 加密技术的卷中的数据的访问权限。
