# Enable-BCLocal

## 摘要
启用BranchCache服务，并使其处于本地缓存模式（local caching mode）。

## 语法

```
Enable-BCLocal [-Force] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
`Enable-BCLocal` cmdlet 可以将 BranchCache 服务设置为本地缓存模式。在本地缓存模式下，客户端会将数据存储在 BranchCache 缓存中，并尝试从本地 BranchCache 缓存中读取数据，但不会与其它节点进行通信，也不会尝试连接到托管的缓存服务器。

## 示例

#### 示例 1：在本地缓存模式下启用 BranchCache
```
PS C:\> Enable-BCLocal
```

此命令启用BranchCache功能，并配置其使用本地缓存模式。
