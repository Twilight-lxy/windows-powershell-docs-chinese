# 启用 BC 下降级功能

## 摘要
用于指示一台正在以降级模式运行操作系统的客户端计算机。

## 语法

```
Enable-BCDowngrading [[-Version] <PreferredContentInformationVersion>] [-Force] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Enable-BCDowngrading` cmdlet 指示运行较新版本 Windows 操作系统的客户端计算机以降级模式运行。在该模式下，客户端会向内容服务器发送针对旧版本的请求，而内容服务器则会响应并向客户端发送相应版本的内容信息。

注意：基于 Windows® 7 的 BranchCache 客户端计算机不受此设置的影响。

## 示例

#### 示例 1：在降级模式下运行
```
PS C:\> Enable-BCDowngrading
```

该命令指示客户端计算机以降级模式运行。
