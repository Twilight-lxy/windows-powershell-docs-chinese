# Set-BCAuthentication

## 摘要
指定 BranchCache 计算机身份验证模式。

## 语法

```
Set-BCAuthentication [-Mode] <ClientAuthenticationMode> [-Force] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Set-BCAuthentication` cmdlet 用于指定客户端计算机的身份验证模式。该设置可以在托管的缓存服务器上进行配置，以便在客户端上传数据之前验证其是否属于指定域。

## 示例

### 示例 1：指定域身份验证
```
PS C:\> Set-BCAuthentication -ClientAuthenticationMode Domain
```

此命令配置客户端计算机在从托管的缓存服务器请求内容时使用域身份验证模式。
