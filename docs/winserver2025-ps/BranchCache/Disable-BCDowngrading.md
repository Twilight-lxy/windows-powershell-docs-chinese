# 禁用BC降级功能

## 摘要
禁用内容信息的降级功能，从而防止客户端计算机从内容服务器请求旧版本的内容信息。

## 语法

```
Disable-BCDowngrading [-Force] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 描述
`Disable-BCDowngrading` 这个 cmdlet 会阻止运行较新版本 Windows 操作系统的客户端计算机继续请求旧格式的内容信息。相反，这些客户端计算机会使用新格式来请求内容信息，从而实现更大的带宽节省和应用程序性能提升。

## 示例

### 示例 1：禁用降级模式
```
PS C:\> Disable-BCDowngrading
```

此命令会禁用降级的BranchCache运行模式。
