# 获取 BC 客户端配置信息

## 摘要
检索当前的BranchCache客户端计算机设置。

## 语法

```
Get-BCClientConfiguration [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [<CommonParameters>]
```

## 描述
`Get-BCClientConfiguration` cmdlet 可以检索当前 BranchCache 客户端计算机的配置信息。

## 示例

#### 示例 1：获取当前客户端计算机的设置
```
PS C:\> Get-BCClientConfiguration
```

该命令用于检索当前的BranchCache客户端计算机设置。
