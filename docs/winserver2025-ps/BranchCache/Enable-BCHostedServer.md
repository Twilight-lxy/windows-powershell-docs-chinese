# 启用BCH托管服务器

## 摘要
将 BranchCache 配置为以托管缓存服务器模式运行。

## 语法

```
Enable-BCHostedServer [-Force] [-RegisterSCP] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Enable-BCHostedServer` cmdlet 用于配置 BranchCache 以托管式缓存服务器模式运行。

## 示例

### 示例 1：启用托管缓存模式
```
PS C:\> Enable-BCHostedServer -RegisterSCP
```

此命令启用托管缓存模式，并注册一个服务连接点以实现自动发现。
