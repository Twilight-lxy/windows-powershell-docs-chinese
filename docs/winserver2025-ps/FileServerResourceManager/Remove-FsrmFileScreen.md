# 删除 FsrmFileScreen

## 摘要
删除一个文件屏幕（或界面）。

## 语法

### 查询（cdxml）（默认值）
```
Remove-FsrmFileScreen [-Path] <String> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject（cdxml）
```
Remove-FsrmFileScreen -InputObject <CimInstance[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>]
 [-AsJob] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Remove-FsrmFileScreen` cmdlet 用于从服务器上删除文件筛选规则。删除文件筛选规则后，服务器将不再阻止文件组被保存到你在 `Path` 参数中指定的文件夹中。

## 示例

### 示例 1：移除文件显示界面
```
PS C:\> Remove-FsrmFileScreen -Path "C:\shares"
```

这个命令会从路径 C:\shares 中删除文件屏幕（file screen）相关的内容。
