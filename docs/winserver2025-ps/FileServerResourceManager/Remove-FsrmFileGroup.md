# 删除Fsrm文件组

## 摘要
从服务器上删除文件组。

## 语法

### 查询（cdxml）（默认值）
```
Remove-FsrmFileGroup [-Name] <String[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject（cdxml）
```
Remove-FsrmFileGroup -InputObject <CimInstance[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>]
 [-AsJob] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Remove-FsrmFileGroup` cmdlet 用于从服务器中删除文件组。删除某个文件组后，服务器将无法为该文件组中定义的文件名模式集合创建文件筛选规则（file screens）或文件筛选规则例外项（file screen exceptions）。

## 示例

### 示例 1：删除一个文件组
```
PS C:\> Remove-FsrmFileGroup -Name "Media Files"
```

这个命令会删除名为“Media Files”的文件组。
