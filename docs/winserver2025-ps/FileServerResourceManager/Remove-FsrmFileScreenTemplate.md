# 删除 FsrmFileScreenTemplate

## 摘要
删除文件屏幕模板。

## 语法

### 查询（cdxml）（默认设置）
```
Remove-FsrmFileScreenTemplate [-Name] <String[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject (cdxml)
```
Remove-FsrmFileScreenTemplate -InputObject <CimInstance[]> [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Remove-FsrmFileScreenTemplate` cmdlet 用于删除一个或多个文件筛选模板。当您删除某个文件筛选模板时，使用该模板创建的其他文件筛选设置不会受到影响。

## 示例

### 示例 1：删除文件屏幕模板
```
PS C:\> Remove-FsrmFileScreenTemplate -Name "Stop media files"
```

此命令会删除名为“Stop media files”的文件屏幕模板。
