# FileServer ResourceManager 模块
## 描述
此参考资料提供了所有文件服务器资源管理器（FSRM）cmdlet的描述和语法信息。这些cmdlet按照它们开头所使用的动词（即命令的动作部分）按字母顺序进行排列。
