# Get-FsrmClassification

## 摘要
获取正在运行的文件分类任务的状态。

## 语法

```
Get-FsrmClassification [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [<CommonParameters>]
```

## 描述
`Get-FsrmClassification` cmdlet 用于获取服务器当前正在运行的文件分类进程的状态。该 cmdlet 返回的状态值有：Unknown（未知）、NotRunning（未运行）、Queued（已排队）或 Running（正在运行）。

## 示例

### 示例 1：获取正在运行的分类任务的状态
```
PS C:\> Get-FsrmClassification
```

这个命令用于获取服务器当前正在运行的分类任务的运行状态。
