# 删除Fsrm存储报告

## 摘要
删除存储报告。

## 语法

### 查询（cdxml）（默认值）
```
Remove-FsrmStorageReport [-Name] <String[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject（cdxml）
```
Remove-FsrmStorageReport -InputObject <CimInstance[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>]
 [-AsJob] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
`Remove-FsrmStorageReport` cmdlet 可以从服务器中删除一个或多个存储报告。但是，您无法删除交互式报告或事件报告。

## 示例

### 示例 1：删除存储报告
```
PS C:\> Remove-FsrmStorageReport -Name "Monthly report"
```

这个命令会删除名为“Monthly report”的存储报告文件。
