# HostComputeService 模块
## 描述
本参考资料提供了有关“Host Compute Service”命令集（cmdlets）的描述及其语法信息。下表按照字母顺序列出了这些命令集。

## HostComputeServiceCmdlets
### [Get-ComputeProcess](./Get-ComputeProcess.md)
从 Hyper-V 主机计算服务中获取正在运行的计算系统的列表。

### [停止计算进程](./Stop-ComputeProcess.md)
在Hyper-V主机计算服务中停止正在运行的计算系统。
