# 获取 AdfsClaimsProviderTrustsGroup

## 摘要
获取一个 AD FS 主张提供者信任组。

## 语法

```
Get-AdfsClaimsProviderTrustsGroup [[-Identifier] <String[]>] [<CommonParameters>]
```

## 描述
`Get-AdfsClaimsProviderTrustsGroup` cmdlet 用于获取 Active Directory Federation Services (AD FS) 的声明提供者信任组。
