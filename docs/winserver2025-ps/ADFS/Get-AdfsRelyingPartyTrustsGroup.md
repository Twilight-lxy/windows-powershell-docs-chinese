# Get-AdfsRelyingPartyTrustsGroup

## 摘要
获取一个依赖方信任组（relying party trust group）。

## 语法

```
Get-AdfsRelyingPartyTrustsGroup [[-Identifier] <String[]>] [<CommonParameters>]
```

## 描述
`Get-AdfsRelyingPartyTrustsGroup` cmdlet 可用于获取 Active Directory Federation Services (AD FS) 中的依赖方信任组。
