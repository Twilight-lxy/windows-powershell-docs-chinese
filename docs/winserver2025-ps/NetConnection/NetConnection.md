# NetConnection 模块

## 描述

这个参考资料提供了所有与网络连接（Net Connection）相关的cmdlet的描述和语法信息。这些cmdlet是按照它们开头所使用的动词（即命令行为语）的字母顺序排列的。

## NetConnectionCmdlets

### [Get-NetConnectionProfile](./Get-NetConnectionProfile.md)
获取一个连接配置文件。

### [Set-NetConnectionProfile](./Set-NetConnectionProfile.md)
更改连接配置文件的网络类别。
