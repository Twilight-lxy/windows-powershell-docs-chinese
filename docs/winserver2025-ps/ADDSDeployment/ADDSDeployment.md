# ADDSDeployment 模块

## 说明

以下内容列出了每个 `ADDSDeployment` cmdlet 的名称及其简要描述。

## ADDSDeploymentCmdlets

### [Add-ADDSReadOnlyDomainControllerAccount](./Add-ADDSReadOnlyDomainControllerAccount.md)

创建一个 RODC（域控制器操作主节点）账户，该账户可用于在 Active Directory 中安装 RODC。

### [安装-添加AD域](./Install-ADDSDomain.md)

安装Active Directory域配置。

### [安装-ADDSDomainController](./Install-ADDSDomainController.md)

在 Active Directory 中安装一个域控制器。

### [安装 AD DS 林](./Install-ADDSForest.md)

安装Active Directory森林配置。

### [Test-ADDSDomainControllerInstallation](./Test-ADDSDomainControllerInstallation.md)

仅运行安装 Active Directory 中域控制器所需的先决条件（prerequisites）。

### [Test-ADDSDomainControllerUninstallation](./Test-ADDSDomainControllerUninstallation.md)

运行用于卸载 Active Directory 中域控制器所需的先决条件程序。

### [Test-ADDSDomainInstallation](./Test-ADDSDomainInstallation.md)

运行安装新的 Active Directory 域配置所需的先决条件程序。

### [Test-ADDSForestInstallation](./Test-ADDSForestInstallation.md)

运行安装Active Directory新林所需的先决条件程序。

### [测试 AD DS 仅读域控制器账户创建功能](./Test-ADDSReadOnlyDomainControllerAccountCreation.md)

运行用于添加 RODC 账户所需的先决条件程序。

### [卸载-ADDSDomainController](./Uninstall-ADDSDomainController.md)

在 Active Directory 中卸载一个域控制器。
