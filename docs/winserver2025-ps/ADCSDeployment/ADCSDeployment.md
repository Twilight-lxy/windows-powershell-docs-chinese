# ADCSDeployment 模块

## 说明

本主题简要介绍了用于部署 Active Directory 证书服务（AD CS）的 Windows PowerShell® cmdlets。表格中的每个 cmdlet 都链接到了有关该 cmdlet 的更多信息。

## ADCSDeploymentCmdlets

### [安装 AdcsCertificationAuthority](./Install-AdcsCertificationAuthority.md)

负责安装和配置AD CS（Active Directory Certificate Services）中的“认证机构”（Certification Authority）角色服务。

### [安装 AdcsEnrollmentPolicyWebService](./Install-AdcsEnrollmentPolicyWebService.md)

执行证书注册策略（Certificate Enrollment Policy）Web服务的配置工作。

### [安装 AdcsEnrollmentWebService](./Install-AdcsEnrollmentWebService.md)

执行证书注册Web服务的初始配置。

### [安装 AdcsNetworkDeviceEnrollmentService](./Install-AdcsNetworkDeviceEnrollmentService.md)

安装NDES角色服务。

### [安装 AdcsOnlineResponder](./Install-AdcsOnlineResponder.md)

安装在线响应器服务。

### [安装 AdcsWebEnrollment](./Install-AdcsWebEnrollment.md)

安装证书颁发机构（CA）的网络注册功能。

### [卸载 AdcsCertificationAuthority](./Uninstall-AdcsCertificationAuthority.md)

卸载CA角色服务并删除相应的配置信息。

### [卸载 AdcsEnrollmentPolicyWebService](./Uninstall-AdcsEnrollmentPolicyWebService.md)

卸载“证书注册策略”Web服务。

### [卸载 AdcsEnrollmentWebService](./Uninstall-AdcsEnrollmentWebService.md)

卸载“证书注册”Web服务或其各个单独实例。

### [卸载 AdcsNetworkDeviceEnrollmentService](./Uninstall-AdcsNetworkDeviceEnrollmentService.md)

卸载NDES角色服务。

### [卸载 AdcsOnlineResponder](./Uninstall-AdcsOnlineResponder.md)

卸载“在线响应器”服务。

### [卸载 AdcsWebEnrollment](./Uninstall-AdcsWebEnrollment.md)

卸载“CA Web Enrollment”角色服务。
