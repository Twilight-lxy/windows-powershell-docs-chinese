# Set-DtcDefault

## 摘要
设置默认的交易协调器。

## 语法

```
Set-DtcDefault [-ServerName <String>] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [<CommonParameters>]
```

## 描述
`Set-DtcDefault` cmdlet 用于设置默认的事务协调器。

## 示例

### 示例 1：设置默认的 DTC 协调器
```
PS C:\> Set-DtcDefault -ServerName "host14"
```

此命令将默认的 DTC 协调器设置为 host14。
