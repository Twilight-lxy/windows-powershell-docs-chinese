# 重置 DTC 日志

## 摘要
重置 DTC 实例的日志文件。

## 语法

```
Reset-DtcLog [-DtcName <String>] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 描述
`Reset-DtcLog` cmdlet 用于重置分布式事务协调器（DTC）实例的日志文件。您可以使用 `DtcName` 参数来指定特定的 DTC 实例。该 cmdlet 还会使用现有的设置重新创建事务日志，并重启相应的 DTC 实例。

## 示例

#### 示例 1：重置 DTC 日志
```
PS C:\> Reset-DtcLog -DtcName "Local" -Confirm:$False -WhatIf:$False
```

此命令会重置本地DTC实例的日志文件。
