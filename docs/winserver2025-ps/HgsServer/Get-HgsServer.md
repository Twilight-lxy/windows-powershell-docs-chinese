# Get-HgsServer

## 摘要
获取Host Guardian服务服务器的状态。

## 语法

```
Get-HgsServer [<CommonParameters>]
```

## 描述
`Get-HgsServer` cmdlet 可以获取 Host Guardian Service (HGS) 服务器的状态，包括密钥保护 URL、认证 URL 以及认证操作模式。

## 示例

#### 示例 1：获取 Host Guardian 服务服务器的状态
```
PS C:\> Get-HgsServer
```

这个命令用于获取Host Guardian Service服务器的状态信息。

此 cmdlet 支持以下常用参数：-Debug、-ErrorAction、-ErrorVariable、-InformationAction、-InformationVariable、-OutVariable、-OutBuffer、-PipelineVariable、-Verbose、-WarningAction 和 -WarningVariable。有关更多信息，请参阅关于(CommonParameters)（http://go.microsoft.com/fwlink/?LinkID=113216）。
