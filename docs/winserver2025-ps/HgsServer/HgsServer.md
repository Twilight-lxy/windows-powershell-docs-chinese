# HgsServer 模块
## 描述
本参考资料提供了Host Guardian Service服务器相关cmdlet的描述和语法信息。下表按照字母顺序列出了所有这些cmdlet。

## HgsServerCmdlets
### [Clear-HgsServer](Clear-HgsServer.md)
将本地节点上的Host Guardian服务重置为未初始化的状态。

### [Export-HgsServerState](Export-HgsServerState.md)
将本地Host Guardian服务实例的状态导出，以便在需要恢复的情况下使用。

### [Get-HgsServer](Get-HgsServer.md)
获取Host Guardian服务服务器的状态。

### [Import-HgsServerState](Import-HgsServerState.md)
将导出的Host Guardian Service状态导入到一个Host Guardian Service实例中。

### [ Initialize-HgsServer](Initialize-HgsServer.md)
初始化Host Guardian服务服务器。

### [安装 HGS 服务器](Install-HgsServer.md)
安装Host Guardian服务服务器。

### [Set-HgsServer](Set-HgsServer.md)
用于设置Host Guardian服务服务器的配置信息。

### [Test-HgsServer](Test-HgsServer.md)
测试本地计算机是否可以作为Host Guardian Service服务器节点运行。

### [卸载 HgsServer](Uninstall-HgsServer.md)
从Host Guardian服务中以及该域名中删除一个本地节点。
