# 清除 Clear-DnsClientCache 的缓存

## 摘要
清空DNS客户端缓存中的内容。

## 语法

```
Clear-DnsClientCache [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
`Clear-DnsClientCache` cmdlet 会删除 DNS 客户端缓存中的所有内容。运行此 cmdlet 的效果等同于执行 `ipconfig /flushdns` 命令。

## 示例

### 示例 1：清除 DNS 客户端缓存
```
PS C:\> Clear-DnsClientCache
```

这个命令会清除DNS客户端缓存。
