# 防御模块

## 描述

这个参考资料提供了所有针对 Defender 的函数的描述和语法信息。它按照函数开头所使用的动词（即函数的行为类型）对这些函数进行了字母顺序的排列。

> [!注意] > 你可能还会听到这些函数被称为“cmdlet”。尽管它们实际上是作为PowerShell函数开发的，但它们的设计目的是使其看起来像cmdlet。

## Defender cmdlets

### [Add-MpPreference](./Add-MpPreference.md)

修改Windows Defender的设置。

### [Get-MpComputerStatus](./Get-MpComputerStatus.md)

获取计算机上反恶意软件软件的状态。

### [Get-MpPreference](./Get-MpPreference.md)

获取Windows Defender扫描和更新的偏好设置。

### [Get-MpThreat](./Get-MpThreat.md)

获取计算机上检测到的威胁的历史记录。

### [Get-MpThreatCatalog](./Get-MpThreatCatalog.md)

从定义目录中获取已知的威胁信息。

### [Get-MpThreatDetection](./Get-MpThreatDetection.md)

该系统能够激活并有效应对Windows Defender检测到的恶意软件威胁。

### [Remove-MpPreference](./Remove-MpPreference.md)

移除排除项或默认操作。

### [Remove-MpThreat](./Remove-MpThreat.md)

从计算机中清除活跃的威胁。

### [Set-MpPreference](./Set-MpPreference.md)

配置 Windows Defender 扫描和更新的偏好设置。

### [开始使用 MpScan](./Start-MpScan.md)

在计算机上开始扫描操作。

### [Start-MpWDOScan](./Start-MpWDOScan.md)

启动Windows Defender的离线扫描功能。

### [Update-MpSignature](./Update-MpSignature.md)

更新计算机上的反恶意软件定义。
