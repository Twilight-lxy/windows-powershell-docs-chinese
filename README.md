# windows powershell 文档中文翻译

基于 [windows-powershell-docs](https://github.com/MicrosoftDocs/windows-powershell-docs) 仓库（2025.12）中的 windows-powershell 文档翻译，大部分内容使用 hunyuan-mt-7b (Q4_K_M) 翻译。

由于内容较多，会逐步翻译所有文档，优先从 winserver2025-ps ，即 Windows Server 2025 and Windows 11 PowerShell modules 开始。

欢迎为未涉及到的翻译内容提供中文翻译。

/docs 目录保存翻译后的markdown文件

/translate 目录保存对应文件的 .po 翻译文件

/src 目录保存应用 .po 文件对原始文件翻译的 python 脚本。
